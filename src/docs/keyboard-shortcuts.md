# Keyboard Shortcuts

All shortcuts use **Ctrl** on Windows/Linux (or **Cmd** on macOS).

## Session Management

| Shortcut | Action |
|----------|--------|
| **Ctrl+N** | Create a new session |
| **Ctrl+W** | Stop the active session |

## Navigation

| Shortcut | Action |
|----------|--------|
| **Ctrl+1** through **Ctrl+9** | Switch to session by position |
| **Ctrl+ArrowDown** | Next session |
| **Ctrl+ArrowUp** | Previous session |

## Interface

| Shortcut | Action |
|----------|--------|
| **Ctrl+B** | Toggle sidebar visibility |
| **Ctrl+,** | Open settings |
| **Ctrl+/** | Show keyboard shortcuts dialog |

## Terminal

All other keyboard input is passed directly to the active terminal session. Tether does not intercept or modify terminal keystrokes -- what you type goes straight to Claude Code.
